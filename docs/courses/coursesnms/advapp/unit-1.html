الأهداف التدريبية

بنهاية هذه الوحدة ستكون قادرًا على:
  • دمج المستندات المختلفة ومقارنتها باستخدام أدوات Word المتقدمة
  • حماية المستندات بكلمات مرور وتقييد التحرير
  • إنشاء وإدارة القوالب والمستندات الرئيسية
  • إدارة المراجع والفهارس التلقائية

1. مقارنة ودمج المستندات

المفاهيم الأساسية:
  • استخدام ميزة Compare لمقارنة نسختين من نفس المستند
  • استخدام ميزة Combine لدمج التعديلات من عدة مستندات
  • تحليل نتائج المقارنة (التعديلات، الحذف، الإضافات)
تمرين عملي:

المهمة: دمج مستند "تقرير التدريب" مع ملاحظات من 3 مشرفين مختلفين

  1. افتح ملف التقرير_الأصلي.docx
  2. استخدم Review > Compare > Combine
  3. أضف ملفات الملاحظات (ملاحظات_أحمد.docx، ملاحظات_سارة.docx)
  4. احفظ النتيجة باسم التقرير_النهائي.docx
نصيحة: استخدم خاصية Track Changes لمتابعة التعديلات

2. حماية المستندات

كلمات المرور

إعداد حماية مزدوجة:

  • كلمة مرور للفتح
  • كلمة مرور للتعديل
تقييد التحرير

تحديد مناطق قابلة للتعديل فقط:

  1. Review > Restrict Editing
  2. تحديد أقسام معينة
تطبيق عملي:

السيناريو: حماية نموذج طلب إجازة

  1. افتح طلب_إجازة.docx
  2. اضغط File > Protect Document
  3. اختر Encrypt with Password
