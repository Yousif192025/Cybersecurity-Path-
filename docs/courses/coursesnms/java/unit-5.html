مدخل إلى إدخال البيانات في جافا

في الوحدات السابقة، كنا نكتب الكود ثم ننفذه مباشرة، بحيث تكون القيم التي نعتمد عليها في البرنامج محددة مسبقًا داخل الكود نفسه. أي أننا كنا نعرف النتيجة قبل تشغيل البرنامج لأننا قمنا بتخزين القيم في المتغيرات أثناء كتابة الكود.

لكن البرمجة التفاعلية تتطلب شيئًا آخر: نريد أن ينفذ البرنامج أوامر معينة بناءً على مدخلات المستخدم، أي أن المستخدم هو من يحدد القيم أثناء تشغيل البرنامج.

في هذه الوحدة سنتعلم كيفية جعل برنامج جافا يتفاعل مع المستخدم، بحيث يطلب منه إدخال بيانات من لوحة المفاتيح (Keyboard)، ثم يقوم البرنامج بمعالجة هذه البيانات أو استخدامها في تنفيذ عمليات معينة.

---

الكلاس Scanner

الكلاس Scanner هو أحد الكلاسات الجاهزة في مكتبة Java. وظيفته الأساسية هي استقبال البيانات من المستخدم (Input) وتحويلها إلى النوع المناسب من البيانات (int, float, String… إلخ) لمعالجتها.

هذا الكلاس واسع الاستخدام، حيث يتكون من 9 كونستركتورات (Constructors) وأكثر من 50 دالة (Methods)، تسمح بمرونة كبيرة في التعامل مع المدخلات.

---

أنواع البيانات التي يمكن استقبالها باستخدام Scanner

يمكن للـ Scanner استقبال معظم أنواع البيانات الأساسية في Java، ومنها:

  • الأعداد الصحيحة (int): مثل 123.
  • الأعداد العشرية (float/double): مثل 10.55.
  • الحروف (char): مثل 'a' أو '3' أو '@'.
  • الكلمات (String): مثل "harmash".
  • الجمل (Strings متعددة الكلمات): مثل: "harmash.com is one of the best sites to learn programming".
  • القيم المنطقية (boolean): مثل true أو false.

إذاً، لكل نوع بيانات تريد استقباله في البرنامج له دالة خاصة ستتعرف عليها لاحقاً.

---

مصطلحات تقنية مهمة

  • Delimiter: هو النمط (Pattern) المستخدم للفصل بين البيانات أو الكلمات، مثل المسافات أو علامات الترقيم. يمكنك برمجيًا تحديد رموز أو أحرف معينة لتُعامل كمسافات فارغة.
  • White Space: تشير إلى الفراغات أو المسافات الفاصلة بين الكلمات (مثل المسافة، التاب، السطر الجديد).
  • Console Application: هو البرنامج الذي يعمل في واجهة نصية فقط، دون واجهة رسومية (GUI). جميع البرامج التي كتبناها سابقًا هي تطبيقات Console لأنها تتفاعل عبر شاشة الإخراج (Console).
---

خطوات جعل البرنامج يستقبل بيانات من المستخدم

لكي نكتب برنامجًا في Java يستقبل بيانات من المستخدم، علينا اتباع ثلاث خطوات رئيسية:

1. استدعاء الكلاس Scanner

هذه الخطوة تتيح لنا استخدام الكلاس Scanner في البرنامج. يجب وضع هذا السطر في بداية ملف الجافا الخاص بك، عادةً بعد حزمة (`package`) البرنامج وقبل تعريف الكلاس.

import java.util.Scanner;

2. إنشاء كائن من الكلاس Scanner

نحن بحاجة لإنشاء كائن (Object) من الكلاس Scanner ليكون مسؤولاً عن استقبال المدخلات. نمرر System.in كمعامل لكونستركتور Scanner، والذي يعني أن مصدر الإدخال سيكون لوحة المفاتيح.

Scanner input = new Scanner(System.in);
// هنا أنشأنا كائن باسم input من الكلاس Scanner، ليكون مسؤولاً عن استقبال المدخلات من لوحة المفاتيح.

3. استخدام إحدى دوال Scanner لإدخال البيانات

بعد إنشاء الكائن، يمكننا استدعاء إحدى دوال الإدخال المتاحة في الكلاس Scanner لاستقبال النوع المحدد من البيانات من المستخدم.

int a = input.nextInt();
// هذا الكود يجعل البرنامج ينتظر من المستخدم إدخال عدد صحيح، ثم يخزن هذه القيمة في المتغير a.
ملاحظة مهمة:

من الأفضل دائمًا وضع دوال الإدخال داخل كتلة `try-catch` لمعالجة أي أخطاء محتملة قد تحدث إذا أدخل المستخدم بيانات من نوع غير متوقع. كما يجب إغلاق الكائن Scanner بعد الانتهاء من استخدامه لتحرير الموارد، وذلك باستخدام الدالة input.close();.

---

أمثلة شاملة حول إدخال البيانات

المثال الأول: برنامج يطلب من المستخدم إدخال اسمه، عمره، ومهنته، ثم يعرض هذه البيانات على الشاشة.

هنا سنستخدم دوال مثل nextLine() و nextInt() من الكلاس Scanner.

import java.util.Scanner; // استيراد الكلاس Scanner

public class UserDataCollector {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in); // إنشاء كائن Scanner

        System.out.print("أدخل اسمك: "); // طلب الاسم
        String name = input.nextLine(); // استقبال سطر كامل (الاسم)

        System.out.print("أدخل عمرك: "); // طلب العمر
        int age = input.nextInt(); // استقبال عدد صحيح (العمر)

        // تنبيه: nextInt() لا تستهلك السطر الجديد، مما قد يؤثر على nextLine() اللاحقة
        // لذا، نضيف input.nextLine() إضافية لامتصاص السطر الجديد العالق
        input.nextLine(); 

        System.out.print("أدخل مهنتك: "); // طلب المهنة
        String profession = input.nextLine(); // استقبال سطر كامل (المهنة)

        System.out.println("\n--- بيانات المستخدم ---");
        System.out.println("الاسم: " + name);
        System.out.println("العمر: " + age + " سنة");
        System.out.println("المهنة: " + profession);

        input.close(); // إغلاق كائن Scanner
    }
}

المثال الثاني: برنامج يطلب من المستخدم إدخال علاماته المدرسية، ثم يخزنها داخل مصفوفة (Array).

هذا يتطلب معرفة بالمصفوفات (سيتم تغطيتها بتفصيل في وحدات لاحقة)، ولكن هذا مثال يوضح إمكانية إدخال عدة قيم.

import java.util.Scanner; // استيراد الكلاس Scanner

public class StudentGrades {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in); // إنشاء كائن Scanner

        System.out.print("كم عدد المواد الدراسية؟ ");
        int numSubjects = input.nextInt();
        input.nextLine(); // امتصاص السطر الجديد

        // تعريف مصفوفة لتخزين الدرجات
        double[] grades = new double[numSubjects];

        System.out.println("الرجاء إدخال علامات المواد:");
               for (int i = 0; i < numSubjects; i++) {
            System.out.print("علامة المادة " + (i+1) + ": ");
            grades[i] = input.nextDouble();
            input.nextLine(); // امتصاص السطر الجديد
        }

        // حساب المعدل
        double sum = 0;
        for (double grade : grades) {
            sum += grade;
        }
        double average = sum / numSubjects;

        System.out.println("\n--- النتائج ---");
        System.out.println("عدد المواد: " + numSubjects);
        System.out.println("المعدل: " + average);

        input.close(); // إغلاق كائن Scanner
    }
}

المثال الثالث: برنامج لتحويل درجة الحرارة من فهرنهايت إلى مئوية

                    
هذا المثال يوضح كيفية أخذ مدخلات عددية وإجراء عمليات حسابية عليها:

                    
import java.util.Scanner;

public class TemperatureConverter {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("أدخل درجة الحرارة بالفهرنهايت: ");
        double fahrenheit = input.nextDouble();
        
        double celsius = (fahrenheit - 32) * 5/9;
        
        System.out.println(fahrenheit + " فهرنهايت = " + celsius + " مئوية");
        
        input.close();
    }
}

كونستركتورات الكلاس Scanner

                    
يحتوي الكلاس Scanner على عدة كونستركتورات تسمح بقراءة المدخلات من مصادر مختلفة:

                    
                    

                        
                            

                                الكونستركتور
                                الوصف
                            

                        
                        
                            

                                Scanner(File source)
                                يقرأ البيانات من ملف
                            

                            

                                Scanner(InputStream source)
                                يقرأ البيانات من مصدر إدخال (مثل System.in)
                            

                            

                                Scanner(Readable source)
                                يقرأ البيانات من مصدر قابل للقراءة
                            

                            

                                Scanner(ReadableByteChannel source)
                                يقرأ البيانات من قناة بايت قابلة للقراءة
                            

                            

                                Scanner(String source)
                                يقرأ البيانات من سلسلة نصية
                            

                        
                    

                    
                    

                         مثال على استخدام كونستركتور الملف:
                        
import java.io.File;
import java.util.Scanner;

public class FileReaderExample {
    public static void main(String[] args) throws Exception {
        Scanner fileScanner = new Scanner(new File("data.txt"));
        
        while (fileScanner.hasNextLine()) {
            System.out.println(fileScanner.nextLine());
        }
        
        fileScanner.close();
    }
}

أهم دوال الإدخال في الكلاس Scanner

                    
                    
دوال قراءة البيانات الأساسية

                    

                        
                            

                                الدالة
                                الوصف
                                مثال
                            

                        
                        
                            

                                nextInt()
                                قراءة عدد صحيح (int)
                                int age = input.nextInt();
                            

                            

                                nextDouble()
                                قراءة عدد عشري (double)
                                double price = input.nextDouble();
                            

                            

                                nextFloat()
                                قراءة عدد عشري (float)
                                float temp = input.nextFloat();
                            

                            

                                nextLong()
                                قراءة عدد صحيح طويل (long)
                                long population = input.nextLong();
                            

                            

                                nextShort()
                                قراءة عدد صحيح قصير (short)
                                short code = input.nextShort();
                            

                            

                                nextByte()
                                قراءة بايت (byte)
                                byte data = input.nextByte();
                            

                            

                                nextBoolean()
                                قراءة قيمة منطقية (boolean)
                                boolean flag = input.nextBoolean();
                            

                            

                                next()
                                قراءة كلمة واحدة (String)
                                String word = input.next();
                            

                            

                                nextLine()
                                قراءة سطر كامل (String)
                                String line = input.nextLine();
                            

                        
                    

                    
                    

                         ملاحظة:
                        
الفرق بين next() و nextLine() هو أن next() تقرأ الكلمة التالية فقط (تتوقف عند المسافة)، بينما nextLine() تقرأ السطر بأكمله حتى نهاية السطر.

دوال البحث (Searching Methods)

                    
هذه الدوال تسمح لك بالتحقق مما إذا كان هناك بيانات من نوع معين متاحة للقراءة:

                    
                    

                        
                            

                                الدالة
                                الوصف
                                مثال
                            

                        
                        
                            

                                hasNext()
                                تتحقق مما إذا كان هناك المزيد من الكلمات
                                if (input.hasNext()) { ... }
                            

                            

                                hasNextInt()
                                تتحقق مما إذا كان التالي عدد صحيح
                                if (input.hasNextInt()) { ... }
                            

                            

                                hasNextDouble()
                                تتحقق مما إذا كان التالي عدد عشري
                                if (input.hasNextDouble()) { ... }
                            

                            

                                hasNextLine()
                                تتحقق مما إذا كان هناك سطر آخر
                                if (input.hasNextLine()) { ... }
                            

                        
                    

                    
                    

                         مثال على استخدام دوال البحث:
                        
Scanner input = new Scanner(System.in);
System.out.print("أدخل عددًا صحيحًا: ");

if (input.hasNextInt()) {
    int number = input.nextInt();
    System.out.println("لقد أدخلت: " + number);
} else {
    System.out.println("لم تدخل عددًا صحيحًا!");
}

input.close();

دوال التحقق (Checking Methods)

                    
هذه الدوال تسمح لك بالتحقق من خصائص الـ Scanner:

                    
                    

                        
                            

                                الدالة
                                الوصف
                                مثال
                            

                        
                        
                            

                                ioException()
                                تُرجع آخر استثناء حدث
                                IOException e = input.ioException();
                            

                            

                                radix()
                                تُرجع أساس نظام الأعداد المستخدم
                                int radix = input.radix();
                            

                            

                                delimiter()
                                تُرجع نمط الفاصل المستخدم
                                Pattern p = input.delimiter();
                            

                            

                                locale()
                                تُرجع الإعدادات المحلية
                                Locale loc = input.locale();

دوال المعالجة (Processing Methods)

                    
هذه الدوال تسمح لك بتعديل سلوك الـ Scanner:

                    
                    

                        
                            

                                الدالة
                                الوصف
                                مثال
                            

                        
                        
                            

                                useDelimiter(String pattern)
                                تحديد نمط الفاصل بين البيانات
                                input.useDelimiter(",");
                            

                            

                                useRadix(int radix)
                                تحديد أساس نظام الأعداد
                                input.useRadix(16); // للنظام الست عشري
                            

                            

                                useLocale(Locale locale)
                                تحديد الإعدادات المحلية
                                input.useLocale(Locale.US);
                            

                            

                                skip(String pattern)
                                تخطي البيانات التي تطابق النمط
                                input.skip("\\s+"); // تخطي المسافات
                            

                            

                                close()
                                إغلاق الـ Scanner
                                input.close();
                            

                        
                    

                    
                    

                         مثال على استخدام الفواصل المخصصة:
                        
Scanner input = new Scanner(System.in);
input.useDelimiter(","); // استخدام الفاصلة كفاصل بين المدخلات

System.out.println("أدخل 3 أرقام مفصولة بفواصل (مثل: 10,20,30): ");
int a = input.nextInt();
int b = input.nextInt();
int c = input.nextInt();

System.out.println("المجموع: " + (a + b + c));
input.close();

الخلاصة

                    

                        
                        
في هذه الوحدة تعلمنا كيفية استخدام الكلاس Scanner لاستقبال مدخلات المستخدم في برامج جافا. إليك النقاط الرئيسية:

                        
    
                            
  • يجب استيراد الكلاس Scanner من الحزمة java.util قبل استخدامه.
    • 
                            
  • نقوم بإنشاء كائن من الكلاس Scanner باستخدام new Scanner(System.in).
    • 
                            
  • لكل نوع بيانات دالة خاصة لقراءته (مثل nextInt() للأعداد الصحيحة).
    • 
                            
  • يجب الانتباه إلى الفرق بين next() و nextLine().
    • 
                            
  • من الجيد استخدام دوال التحقق مثل hasNextInt() قبل قراءة المدخلات.
    • 
                            
  • يجب إغلاق الـ Scanner باستخدام close() عند الانتهاء منه.
    • 
                        

                    

                    
                    

                         نشاط عملي:
                        
قم بإنشاء برنامج يحسب مساحة المستطيل حيث يقوم المستخدم بإدخال الطول والعرض. تأكد من التحقق من أن المدخلات صحيحة (أعداد موجبة) قبل إجراء الحساب.
