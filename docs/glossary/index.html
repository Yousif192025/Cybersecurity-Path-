التصيد (Phishing)

هجوم إلكتروني يتم عبر رسائل مزيفة تظهر كأنها من جهات موثوقة، بهدف خداع المستخدمين للحصول على معلومات حساسة مثل كلمات المرور أو بيانات البطاقات الائتمانية.

المزيد

البرمجيات الخبيثة (Malware)

أي برنامج مصمم لإلحاق الضرر بأنظمة الحاسوب أو الشبكات أو المستخدمين، ويتضمن الفيروسات، أحصنة طروادة، برامج الفدية، وبرامج التجسس.

المزيد

جدار الحماية (Firewall)

نظام أمان للشبكة يراقب حركة مرور الشبكة الواردة والصادرة ويقرر السماح أو حظر حركة المرور بناءً على مجموعة محددة من قواعد الأمان.
